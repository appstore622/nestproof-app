Getting started

How to use NestProof

  • Add documents from the Documents tab (camera, photo library, or files).
  • Track items under Inventory with rooms, serial numbers, and barcodes.
  • Use Search to find OCR text, serial numbers, and notes.
  • Enable Face ID or passcode under Settings → Security.

Export & backup

Data management

  • Insurance Report (PDF) — Settings → Data → Export Insurance Report.
  • CSV / JSON / ZIP — Settings → Data for spreadsheet or full backup.
  • Encrypted iCloud backup — optional; encrypts on device before saving to iCloud Drive.

FAQ

Common questions

Do I need an account?

No. NestProof works offline with no login.

Is there a subscription?

No. NestProof is a one-time App Store purchase.

Barcode scan not working?

Requires a physical iPhone with camera; simulators do not support scanning.

Restore from backup?

Settings → Data → Import Backup, or Encrypted Backup for password-protected restore.

App Review

Testing notes

NestProof is a paid, offline-first private vault. No login, server, or demo account is required.

  • Launch the app and complete onboarding (optional Face ID lock).
  • Add a document via camera or photo import; OCR runs on-device.
  • Add an inventory item; barcode scan requires a physical device with camera.
  • Settings → Data → Export Insurance Report (PDF) to verify export.
  • Settings → Security to test Face ID / passcode lock.

Encrypted iCloud backup is optional and user-initiated. No third-party analytics or ads. Bundle ID: com.nestproof.vault.
